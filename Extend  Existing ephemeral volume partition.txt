# Extend  Existing ephemeral volume partition ----
@ First upgrade volume from console
1) lsblk  to check the volume size

2)  lsblk -f  to check filesystem type

3) sudo growpart /dev/xvdf 1 or 2 or 3 or 4 ( partition)

4) now resize file system----

sudo resize2fs /dev/xvdf1
or
sudo xfs_growfs /dev/xvdf1

4) Now check filesystem

df -h
